Custom Theme Example

This window demonstrates custom theming capabilities in Wails v3.

Open the "Themes" menu to create new windows with different themes.

Available Themes:

🌊 Midnight Ocean - A deep, calming blue theme

🌲 Forest Haven - An elegant green theme

👑 Royal Purple - A sophisticated purple theme

🏜️ Desert Sand - A warm, neutral theme

❄️ Arctic Frost - A clean, crisp light theme
